---
title: "text-shadow"
type: docs
description: Text shadow property
---

Sets whether the text has a shadow.

| Field | Value |
|--|--|
| Cascades | Yes |
| Default value | `false` |

## Values
Accepts any expression or value that resolves to a boolean value.

## Examples
```scss
p {
  text-shadow: false;
}
p {
  text-shadow: true;
}
```
