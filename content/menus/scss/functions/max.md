---
title: "max(a, b)"
type: docs
description: "Returns the largest of the values specified"
---
```scss
@function max($a, $b)
```

## Parameters
- **a**  
  The first value
- **b**  
  The second value

## Returns
Either the `a` or `b`. `a` is returned if it is greater than `b`, otherwise `b` 
is returned.
